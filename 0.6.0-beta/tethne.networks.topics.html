tethne.networks.topics.topic_coupling(model, papers=None, threshold=None)[source]¶

Builds a network of topics using inverse symmetric KL-divergence on papers.

If papers is not None, uses only those papers provided to calculate KL-divergence.

Parameters:

model : LDAModel

papers : list

A list of paper indices to use in KL-divergence calculation.

threshold : float

Minimum inverse symmetric KL-divergence for an edge. (default = 0.25)
