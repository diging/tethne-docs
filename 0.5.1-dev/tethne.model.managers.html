Module contents¶

This module provides manager classes for generating, visualizing, and analyzing models.

The following managers are presently available:

dtm.DTMModelManager(D[, feature, outpath, ...]) Generates a DTMModel from a Corpus using Gerrish’s C++ implementation.
mallet.MALLETModelManager(D[, feature, ...]) Generates a LDAModel from a Corpus using MALLET.
tap.TAPModelManager([D, G, model]) Generates a time-sensitive set of TAPModels from a Corpus, a corpus model, and a coauthorship GraphCollection.

More managers will be added regularly.
