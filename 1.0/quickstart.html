Quickstart¶

Load some data¶

Assuming that you have a JSTOR DfR dataset (in XML format) containing some wordcount data unzipped at /path/to/my/dataset, create a Corpus with:

>>> from tethne.readers import dfr
>>> dfr_corpus = dfr.read('/path/to/my/dataset')

Or if you’re working with data from the Web of Science, try:

>>> from tethne.readers import wos
>>> wos_corpus = dfr.read('/path/to/my/wosdata.txt')

Your corpus is automatically indexed by author, publication date, and (if available) citations.

>>> corpus.indices.keys()
['date', 'citations', 'authors']

You can select all of the Papers published in 1980 with:

>>> corpus[('date', 1980)]

 <tethne.classes.paper.Paper at 0x106c7b050>,
 ...

 <tethne.classes.paper.Paper at 0x10718b490>,
 <tethne.classes.paper.Paper at 0x10695b850>,

Create a new index using the Corpus.index() method:

>>> corpus.index('journal')
>>> corpus.indices.keys()
['date', 'journal', 'citations', 'authors']

>>> corpus[('journal', 'Journal of the History of Biology')]

Corpus.distribution() will show you how the Papers in your Corpus are distributed over time.

>>> x, y = corpus.distribution()
>>> import matplotlib.pyplot as plt
>>> plt.bar(x, y)

Simple networks simply¶

Network-building methods are available in tethne.networks. You can create a coauthors() network like this:

>>> from tethne.networks import coauthors
>>> coauthor_graph = coauthors(wos_corpus)

All of Tethne’s network-building methods return networkx.Graph objects. For more information, see the NetworkX documentation.

To create a time-variant coauthor network, use a GraphCollection.

>>> from tethne import GraphCollection
>>> coauthor_collection = GraphCollection(wos_corpus, coauthors)
>>> coauthor_collection.node_distribution()
{1980: 32,
 1981: 26,
 1982: 24,
 1983: 26,
 1984: 20,
 1985: 30,
 1986: 35,
 1987: 49,
 1988: 60,
 1989: 66,
 1990: 69,
 1991: 85,
 1992: 82,
 1993: 91,
 1994: 110}

You access the network for 1992 like this:

>>> coauthor_collection[1992]
<networkx.classes.graph.Graph at 0x15f3016d0>

You can control how the GraphCollection is assembled by passing slice_kwargs to the constructor. To create a series of coauthor graphs using a 4-year sliding time-window, for example, you could do:

>>> from tethne import GraphCollection
>>> coauthor_collection = GraphCollection(wos_corpus, coauthors,            \
                                          slice_kwargs={'window_size': 4,   \
                                                        'step_size': 1})
>>> coauthor_collection.node_distribution()
{1980: 99,
 1981: 90,
 1982: 92,
 1983: 100,
 1984: 117,
 1985: 151,
 1986: 180,
 1987: 209,
 1988: 241,
 1989: 261,
 1990: 287,
 1991: 321}

If you’re using WoS data (with citations), you can also build citation-based graphs (see networks.papers). Here’s a static co-citation graph from a Corpus:

>> from tethne.networks import cocitation
>>> cocitation_graph = cocitation(wos_corpus, min_weight=3)

min_weight=3 means that a pair of papers must be co-cited three times to be included in the network.

Visualize your networks¶

You can export a graph for visualization in Cytoscape or Gephi using tethne.writers:

>>> from tethne.writers.graph import to_graphml
>>> to_graphml(coauthor_graph, '/path/to/my/graph.graphml')

To visualize a GraphCollection as a dynamic graph in Cytoscape, export it using writers.collection.to_dxgmml():

>>> from tethne.writers.collection import to_dxgmml
>>> to_dxgmml(coauthor_collection, '/path/to/my/dynamicNetwork.xgmml')

Working with Words¶

Suppose you loaded up a Corpus from some DfR datasets, using:

>>> from tethne.readers import dfr
>>> dfr_corpus = dfr.read('/path/to/my/dataset')

Now you have some 'wordcounts' in dfr_corpus.features.

>>> dfr_corpus.features
{'authors': <tethne.classes.feature.FeatureSet at 0x100534e90>,
 'bigrams': <tethne.classes.feature.FeatureSet at 0x11e599810>,
 'citations': <tethne.classes.feature.FeatureSet at 0x10051b990>,
 'keyterms': <tethne.classes.feature.FeatureSet at 0x107355ad0>,
 'trigrams': <tethne.classes.feature.FeatureSet at 0x14c126050>,
 'wordcounts': <tethne.classes.feature.FeatureSet at 0x1931c7290>}

You can retrieve the wordcounts for an individual paper using it’s DOI:

>>> dfr_corpus.features['wordcounts'].features['10.2307/2486258']
[('secondly', 4),
 ('ainalysis', 2),
 ('limited', 2),
 ('dynamic', 16),
 ('externally', 1),
 ('foul', 1),
 ('four', 3),
 ('demanded', 1),
 ('relationships', 9),
 ('whose', 2),
 ('capolitical', 1),
 ('coalignmnent', 1),
 ('presents', 3),
 ('investigation', 1),
 ('systemns', 1),
 ('admninistration', 1),
 ('conjecture', 1),
 ... ]

You can apply a stoplist using FeatureSet.transform().

>>> from nltk.corpus import stopwords
>>> stoplist = set(stopwords.words())
>>> apply_stoplist = lambda f, v, C, DC: None if f in stoplist else v
>>> wordcounts = dfr_corpus.features['wordcounts']
>>> dfr_corpus.features['filtered'] = wordcounts.transform(apply_stoplist)

If you have some recent WoS data with abstracts, you can get a FeatureSet from abstract terms, too:

>>> from tethne.readers import wos
>>> wos_corpus = dfr.read('/path/to/my/wosdata.txt')
>>> from tethne import tokenize
>>> wos_corpus.index_feature('abstract', tokenize=tokenize)
>>> print wos_corpus.features
['abstract', 'authors', 'citations', 'date']

You can see how the word empirical is distributed across your Corpus using :func:`.Corpus.distribution:

>>> x, y = wos_corpus.feature_distribution('wordcounts', 'empirical')
>>> plt.plot(x, y, lw=2)
>>> plt.ylabel('Frequency of the word `empirical`')
>>> plt.show()

Models Based on Words¶

Topic models are pretty popular. You can create a LDA topic model using MALLET right from Tethne, using tethne.model.corpus.mallet.LDAModel.

>>> from tethne import LDAModel
>>> model = LDAModel(wos_corpus, 'abstract')     # Use words from 'abstract'.
>>> model.fit(Z=20, max_iter=500)    # 20 topics, 500 iterations of sampling.

You can inspect the inferred topics using LDAModel.print_topics().

>>> model.print_topics(5)

We can also look at the representation of a topic over time using LDAModel.topic_over_time():

>>> plt.figure(figsize=(15, 5))
>>> for k in xrange(5):
...     x, y = model.topic_over_time(k)
...     plt.plot(x, y, label='topic {0}'.format(k), lw=2, alpha=0.7)
>>> plt.legend(loc='best')
>>> plt.show()
